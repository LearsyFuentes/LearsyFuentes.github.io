GitHub Pages es un servicio de alojamiento para webs estáticas que toma archivos HTML, CSS y Javascript directamente de un repositorio en GitHub y que, opcionalmente, ejecuta los archivos a través de un proceso de compilación y publica un sitio web. Se puede alojar el site en github.io o en tu propio dominio personalizado.
